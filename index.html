Rapport de Stage PFE

Le projet de fin d'étude consiste à héberger un site web sur Azure Cloud. Pour cela, plusieurs options sont possibles. L'une d'entre elles est d'utiliser Azure App Service, qui permet d'héberger des applications web et des API dans Azure de manière simple et rapide. Une autre option est d'activer l'hébergement de site web statique avec le Portail Azure, Azure CLI ou PowerShell. Cette fonctionnalité doit être activée sur le compte de stockage. Le Cloud Computing peut permettre d'effectuer des économies grâce à la mutualisation des services sur un grand nombre de clients. Le projet devra donc prendre en compte les avantages, inconvénients, risques et offres du marché pour élaborer une solution technologique de Cloud Computing répondant aux besoins du site web à héberger.
